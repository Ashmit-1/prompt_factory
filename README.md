# Prompt Factory

A lightweight, browser-native utility for systematically constructing, formatting, and managing structured LLM prompts.

## Features
- **Local-First**: All data is stored in the browser's IndexedDB via \`localForage\`.
- **Minimalist Design**: Strict Black & White theme using Roboto.
- **No Backend**: Pure client-side application.
- **Data Portability**: Export/Import local data as JSON backup.

## Setup Instructions

1. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`

2. Start the development server:
   \`\`\`bash
   npm run dev
